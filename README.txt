# Проект "Global Warming/ Глобальное Потепление"

## Описание проекта
Проект "Global Warming/ Глобальное Потепление" является инициативой, созданной Яшей Лесиным для изучения и борьбы с проблемой глобального потепления. Проект нацелен на сбор данных, создание информационных ресурсов и поощрение устойчивых практик для уменьшения негативного воздействия человечества на природу.

## Основные функции проекта
1. Здесь Максим 100% попадется на риккрол и здесь мы сможем найти много информации о проблемами природы и о глобальном потепление.

3. **Советы по устойчивости:** Пользователи могут получать рекомендации и советы по уменьшению своего экологического следа и принятию устойчивых образцов поведения.

## Технологии и стек
- **Язык программирования:** Python
- **База данных:** SQLAlchemy
- **Фронтенд:** HTML, CSS, Python
